Claude Code supprimé de l'offre Pro d'Anthropic

Titre original : Claude Code removed from Anthropic's Pro plan

Publié sur HN le 2026-04-21 21:03 · Entré dans /best le 2026-04-22 00:19 · Résumé généré le 2026-04-22 00:20

Résumé de l'article

Anthropic a restructuré son offre tarifaire en retirant Claude Code de son plan Pro. Cette fonctionnalité de génération et d'exécution de code est désormais disponible uniquement dans les plans Team et Enterprise.

  • Claude Code n'est plus inclus dans l'offre Pro (anciennement disponible)
  • La fonctionnalité reste accessible aux utilisateurs des plans Team ($25/mois en facturation mensuelle) et Enterprise
  • Le plan Pro conserve Claude Cowork, l'accès à Research et à plusieurs modèles Claude pour $20/mois
  • Les plans Free, Max et les offres éducatives/académiques conservent ou ont accès à Claude Code selon leur tier respectif

Discussion sur Hacker News

Confirmations :

  • Les contraintes de capacité et la croissance insoutenable des usages (agents autonomes 24/7, tokens consommés massivement par Opus 4.7) justifient une révision des plans tarifaires, particulièrement pour les Pro plan à faible coût.
  • Le retrait de Claude Code du plan Pro fait économiquement sens : subsidier des accès illimités à des outils d'agents autonomes coûteux était structurellement non rentable face à la demande galopante.
  • Anthropic teste une réallocation des ressources vers les vrais clients (Max/entreprises payant des milliers/mois) plutôt que les utilisateurs occasionnels, stratégie classique de monétisation SaaS.
  • Ce changement force les utilisateurs sérieux à basculer vers des offres plus chères ou alternatives (Codex, GLM, Kimi), ce qui reflète une réalité : à $20/mois, les utilisateurs ne sont pas des clients viables pour des workloads coûteux.

Réfutations :

  • Anthropic sabote sa propre acquisition client : les utilisateurs Pro ont servi de tremplin vers des upgrades (beaucoup signalent être passés de $20 à $100-200 grâce à Claude Code gratuit), supprimant cette rampe brise le flywheel de croissance.
  • La communication est désastreuse (modifications silencieuses sur les pages de support, A/B test secret sur 2% des utilisateurs) : cela détruit la confiance bien plus que une hausse tarifaire annoncée clairement, poussant les utilisateurs vers les concurrents.
  • Codex ($20 avec bien plus de quotas) et les modèles chinois (Kimi, GLM) offrent de meilleures conditions sans ces drames marketing, capturant les utilisateurs que Claude pousse dehors.
  • L'absence de grandfathering pour les clients existants ayant signé avec Claude Code inclus constitue une violation implicite du contrat, non l'inverse : Anthropic reçoit l'argent, change les termes après coup.
