## Résumé de l'article

Claude Code, l'assistant de codage d'Anthropic, utilise depuis la version 2.1.181 (juin 2026) le portage en Rust de Bun, l'environnement d'exécution JavaScript. Cet changement a permis d'accélérer le démarrage de 10 % sur Linux, mais passe largement inaperçu pour les utilisateurs.

- Claude Code embarque Bun v1.4.0 (une version canary pas encore officiellement publiée), confirmable par inspection des binaires avec des commandes comme `strings` et `grep`
- Le portage en Rust de Bun contient environ 563 fichiers source Rust identifiables dans le binaire et s'exécute en production sur des millions d'appareils
- L'amélioration de démarrage de 10 % sur Linux valide le gain de performance du passage de Zig (langage original) à Rust, bien que l'impact reste transparent pour la plupart des utilisateurs

## Discussion sur Hacker News (312 commentaires)

**Avis positifs** :
- Le rewrite en Rust fonctionne bien en production avec Claude Code depuis plus d'un mois sans régression mesurable identifiée
- La migration démontre que les modèles IA sont désormais capables de transpiler fiablement des millions de lignes de code entre langages
- Rust offre de meilleures garanties de sécurité mémoire que Zig, particulièrement important pour un runtime JavaScript interfaçant du code C
- Le code Zig original était lui-même un port ligne par ligne d'esbuild, donc la stabilité du codebase était déjà questionnable
- Les utilisateurs finaux de Claude Code n'ont remarqué aucune dégradation perceptible ; la stabilité et performance restent acceptables

**Avis négatifs** :
- Anthropic utilise une version 1.4.0 de Bun non publiquement disponible en tant que version stable avant sa sortie officielle, ce qui viole l'esprit de FOSS
- Le rewrite massif sans implication communautaire a invalidé des centaines de PRs ouvertes et aliéné la communauté Zig qui contribuait au projet
- Le code Rust généré reste extrêmement non-idiomatique avec ~14000 blocs unsafe sans réduction significative depuis deux mois, annulant les bénéfices censés de Rust
- Claude Code lui-même souffre de bugs de rendu, corruptions d'historique et fuites mémoire croissantes coïncidant avec cette migration
- La gouvernance du projet est opaque : Anthropic contrôle unilatéralement les décisions sans processus transparent, tuant efficacement le projet open source communautaire
