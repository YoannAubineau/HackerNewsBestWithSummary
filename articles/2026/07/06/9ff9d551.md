## Résumé de l'article

OpenWrt One est un routeur open hardware basé sur le SoC MediaTek Filogic 820, conçu pour fonctionner avec le firmware OpenWrt. Il offre WiFi 6 dual-band, ports Gigabit/2.5 Gigabit, 1 Go de RAM DDR4, stockage NAND et NOR, et prend en charge Power Over Ethernet (POE).

- Matériel : WiFi 6 (3×3/2×2), 1x 2.5 Gbps WAN, 1x 1 Gbps LAN, support M.2 SSD, console USB-C intégrée, alimentation par Ethernet 802.3af/at
- Installation simple : préchargé avec le firmware OpenWrt le plus récent, interface LuCI GUI, accessible via 192.168.1.1 après connexion au port 1G
- Mise à jour firmware : possible via USB (FAT32), NAND recovery mode, ou via TFTP sur port série UART
- Récupération d'urgence : deux modes de démarrage (NAND/NOR) sélectionnables par commutateur, NOR dédié à la récupération en cas de défaut NAND
- Accès console : port USB-C série intégré sur façade, configuration identique sur Windows (PuTTY), Linux et macOS (screen) à 115200 bauds

## Discussion sur Hacker News (121 commentaires)

**Avis positifs** :
- OpenWRT One offre une plateforme de référence stable et entièrement supportée pour les développeurs qui créent des produits et des images personnalisées, sans limitation de cycle de vie des fabricants
- Le matériel ouvert résout le problème de la disponibilité à long terme : contrairement aux routeurs propriétaires qui disparaissent, on peut se procurer des OpenWRT One n'importe où et les usines peuvent faire de petites séries de production
- OpenWRT offre une liberté logicielle supérieure avec des mises à jour illimitées et des fonctionnalités additionnelles gratuites, fermant une backdoor majeure (TR-069) que les routeurs ISP exploitent
- Les utilisateurs témoignent d'une grande satisfaction : stabilité égale ou supérieure aux routeurs commerciaux, configuration facile de contrôles parentaux, mises à jour simples via l'interface web depuis la version 25.12
- Le prix (~85-106 USD) est compétitif pour un appareil entièrement open-source avec développement continu et sans compromis sur la vie privée

**Avis négatifs** :
- Le hardware n'est pas optimisé pour le coût et reste cher comparé aux routeurs bas de gamme OpenWRT existants offrant 4x les capacités au même prix
- La connectivité est limitée : 1 seul port LAN (+ 1 port 2.5GbE WAN), PoE via carte fille surdimensionnée impraticable pour les vrais boîtiers, WiFi 5 seulement (pas WiFi 7)
- Le cas d'usage pour une plateforme de référence est flou : peu de matériel commercial réel utilise la même architecture, donc les développements faits dessus ne se transfèrent pas facilement
- OpenWRT a une courbe d'apprentissage raide pour les utilisateurs non-experts (installations complexes, mises à jour délicates, documentation dispersée), rendant l'investissement en temps considérable
- Le marché laisse déjà de meilleures options : routeurs Dynalink DL-WRX36, Asus BT8 WiFi 7, ou des dispositifs GL.iNet avec plus de ports et de support matériel
