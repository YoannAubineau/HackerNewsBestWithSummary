## Résumé de l'article

LG installe automatiquement via Windows Update une application qui promeut des abonnements McAfee sur les moniteurs connectés à des PC Windows, sans demander de consentement préalable. Le test de GamersNexus a documenté l'installation sur plusieurs modèles de moniteurs, y compris des appareils achetés plusieurs années auparavant.

- Windows Update installe d'abord des composants d'extension LG, suivis automatiquement par le LG Monitor App Installer, sans notification à l'utilisateur
- L'application a affiché des promotions McAfee lors de 31 des 32 démarrages consécutifs testés, proposant une version d'essai gratuite convertible en abonnement payant
- Le comportement affecte au moins les modèles UltraGear 34GX900A-B et UltraFine 32UN880-B, avec des plaintes remontant au moins à 2024
- Dell utilise une méthode similaire pour installer Alienware Command Center via Windows Update
- Les utilisateurs peuvent bloquer cette installation en activant le paramètre de stratégie de groupe "Empêcher le téléchargement automatique d'applications associées aux métadonnées de périphérique"

## Discussion sur Hacker News (224 commentaires)

**Avis positifs** :
- C'est un problème réel et confirmé : plusieurs utilisateurs attestent avoir observé l'installation silencieuse du logiciel LG via Windows Update sans consentement explicite.
- Des solutions techniques existent pour désactiver ce comportement (gpedit.msc, paramètres d'installation des appareils), même si elles demandent une expertise que l'utilisateur moyen ne possède pas.
- Le problème illustre un dysfonctionnement systémique de Windows depuis des années : autorun, drivers packages, et mécanismes d'installation silencieuse exploités par les constructeurs.
- Microsoft porte une responsabilité majeure en permettant cette pratique via Windows Update sans imposer de vérifications strictes sur le contenu des packages.

**Avis négatifs** :
- Blâmer uniquement LG occulte que Microsoft contrôle entièrement le mécanisme d'installation : LG ne peut installer que ce que Microsoft autorise à passer par Windows Update.
- La pratique n'est pas sans précédent : Razer, Logitech, NVIDIA et d'autres constructeurs font cela depuis des années via le même mécanisme Windows.
- Pour l'utilisateur final, les solutions (gpedit.msc, PowerShell, scripts) sont trop techniques et inaccessibles ; c'est déraisonnable d'exiger cela pour un usage normal.
- La culpabilité est partagée : utilisateurs sous Windows acceptent implicitement un modèle où le système d'exploitation n'est pas vraiment le leur, Microsoft permet les abus, et les constructeurs les exploitent.
- Même en Europe avec GDPR, les recours sont complexes et les entreprises s'en tirent sans conséquence réelle, rendant impossible de changer ce comportement par la protestation seule.
