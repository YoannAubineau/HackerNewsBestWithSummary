## Discussion sur Hacker News (150 commentaires)

**Avis positifs** :
- Les étudiants doivent apprendre les fondamentaux et la profondeur technique avant de dépendre des outils automatisés : comprendre comment les choses fonctionnent vraiment crée une base solide pour résoudre les vrais problèmes nouveaux que les LLM ne peuvent pas gérer.
- L'éthique en informatique est chroniquement négligée malgré son importance critère : les cours d'éthique, quand ils existent, restent insuffisants pour contrer les pressions industrielles réelles et les dilemmes moraux auxquels les ingénieurs feront face.
- Le rythme effréné de l'industrie crée une dette technique massive et invisible : refactorisation, documentation et réflexion profonde économisent des années d'heures-personnes futures, ce que les bilans comptables actuels ne reflètent pas.
- Il existe une distinction importante entre réussir carrièrement et vivre selon ses principes : l'académie crée un espace rare pour explorer ces questions avant que les pressions économiques immédiates ne dominent.
- La conscience critique face aux narratifs technologiques inévitables est pertinente : les technologies ne s'imposent pas seules, elles sont des choix collectifs que les ingénieurs peuvent influencer ou refuser.

**Avis négatifs** :
- Ce conseil risque de mener au chômage pour les jeunes diplômés : refactoriser méticuleusement pendant que tout le monde va deux fois plus vite est une recette pour être dépassé et viré.
- L'auteur n'a pas d'expérience industrielle réelle et ignore les contraintes commerciales qui rendent ses idéaux impraticables : polir le code n'a de sens que si le produit peut d'abord exister et générer des revenus.
- La notion que les étudiants peuvent choisir leurs principes moraux dès le départ est naïve : en réalité, les pressions structurelles et les besoins de survie financière forcent les compromis que l'académie permet d'ignorer.
- Cultiver une pensée profonde en évitant les technologies essentielles est un luxe de tenure académique : les nouveaux venus n'ont ni le pouvoir ni la position pour se permettre ce genre de résistance sans conséquences.
- L'argument du 'vegetarienne IA' s'accompagne d'une hypocrisie cachée : refuser les LLMs tout en utilisant l'informatique construite sur l'exploitation revient à se complaire dans une moralité de surface.
