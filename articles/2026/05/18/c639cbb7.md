## Résumé de l'article

Auto-identity-remove est un script Node.js open source (macOS) qui automatise mensuellement la suppression de vos données personnelles auprès de plus de 500 sites de recherche de personnes et bases de données de courtiers en données, avec résolution automatique de CAPTCHAs et notifications iMessage des résultats.

- Le script recherche votre profil sur chaque site, remplit et soumet les formulaires de désinscription automatiquement, et saute les courtiers d'où vous avez déjà été retiré dans les 90 derniers jours
- Installation simple via npm avec configuration interactive (setup.js) qui génère un fichier de configuration gitignorée stockant vos données personnelles localement sans les envoyer ailleurs
- Utilise CapSolver pour résoudre les reCAPTCHA (~$0,001 par résolution) ; les sites non gérés automatiquement s'ouvrent pour action manuelle
- Chaque exécution produit un résumé d'état (Supprimé, Ignoré, Non listé, Manuel requis, Erreur) et stocke l'historique dans state.json pour éviter les renvois inutiles
- Actuellement couvre ~30 courtiers majeurs avec des stratégies de clic/formulaires spécifiques et ~470 sites génériques ; les contributions sont bienvenues pour ajouter de nouveaux courtiers avec sélecteurs vérifiés

## Discussion sur Hacker News (113 commentaires)

**Avis positifs** :
- Automatiser les demandes d'opt-out auprès de centaines de courtiers en données répond à un besoin réel de réduire les appels et SMS de spam
- Publier le projet en open source permet à la communauté de contribuer et d'améliorer l'outil pour en étendre la couverture et la fiabilité
- L'utilisation d'une approche heuristique couplée à des définitions explicites pour les sites majeurs est une stratégie pragmatique face à la diversité des formulaires
- C'est une bonne application des outils d'automatisation et de l'IA pour résoudre un problème concret affectant les utilisateurs
- Le projet aborde un manque dans l'écosystème open source en proposant une alternative aux services payants existants

**Avis négatifs** :
- Le code présente les marques d'une génération par IA, ce qui rend difficile l'évaluation du travail réellement accompli et des limites restantes
- L'outil est actuellement limité à macOS (launchd, Messages) ; sans support Linux/Windows/cron, il exclut la majorité des utilisateurs potentiels
- Il existe un risque paradoxal que soumettre automatiquement vos données personnelles à 500 sites les consolide davantage au lieu de les supprimer, les marquant comme des adresses actives
- De nombreux courtiers en données ignorent délibérément les demandes d'opt-out malgré les formulaires affichés ; cette solution peut être théâtrale sans impact réel
- Les formulaires d'opt-out requièrent souvent une confirmation par email ou des interventions manuelles ; l'outil ne peut pas accéder aux boîtes mail pour cliquer les liens de validation, limitant son efficacité
