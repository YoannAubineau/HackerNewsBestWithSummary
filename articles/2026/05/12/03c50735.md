## Résumé de l'article

Claude Platform on AWS est une offre d'Anthropic permettant aux clients AWS d'accéder à l'ensemble des fonctionnalités de Claude API via l'authentification AWS IAM, la facturation consolidée et les engagements AWS. Contrairement à Claude sur Amazon Bedrock (où AWS reste processeur de données), Claude Platform on AWS est opéré par Anthropic avec traitement des données hors des frontières AWS.

- Accès aux fonctionnalités complètes de Claude Platform, y compris les agents gérés (Claude Managed Agents), l'exécution de code, le web search, la mise en cache des prompts et les outils d'amélioration de prompts via la Console Claude
- Intégration native AWS : authentification via IAM, audit via CloudTrail, facturation sur une seule facture AWS compatible avec les engagements existants
- Disponibilité en régions AWS commerciales majeures avec support des géographies d'inférence globale et US
- Modèles Claude Opus 4.7, Sonnet 4.6 et Haiku 4.5 accessibles avec déploiement des nouveaux modèles au lancement
- Choix entre Claude Platform on AWS (expérience complète) et Claude sur Bedrock (résidence de données stricte dans AWS)

## Discussion sur Hacker News (89 commentaires)

**Avis positifs** :
- Simplifie considérablement les achats pour les grandes organisations en contournant les processus de procuration complexes et les négociations de contrats avec nouveaux vendeurs, grâce à la facturation consolidée via AWS
- Offre une parité complète des fonctionnalités avec la plateforme Claude native dès le premier jour, contrairement à Bedrock qui accuse un retard de plusieurs mois voire un an
- Permet aux équipes d'utiliser l'infrastructure existante d'AWS (IAM, SSO, FinOps) pour gérer l'accès à Claude, réduisant les frictions organisationnelles et les processus d'approbation
- Peut aider les startups à utiliser leurs crédits AWS existants pour accéder à Claude, maximisant la valeur de leurs allocations de crédits
- Rend Claude plus sticky pour les entreprises fortement investies dans AWS, réduisant le risque de migration vers des alternatives comme OpenAI sur Bedrock

**Avis négatifs** :
- L'offre crée une confusion de nommage supplémentaire (Claude Platform on AWS vs Claude on Bedrock) sans bénéfice technique réel, perpétuant un problème récurrent chez AWS
- Bedrock reste plus fiable et complete que cette nouvelle offre (moins de 500 errors, meilleur support communautaire), rendant le passage peu justifié techniquement
- La facturation via AWS Marketplace génère des factures séparées et n'est pas couverte par tous les types de crédits AWS, limitant les avantages de consolidation supposés
- Pour les entreprises avec des exigences strictes de conformité et de résidence des données, cette offre ne résout rien puisque les données sont traitées en dehors d'AWS contrairement à Bedrock
- Anthropic n'a pas encore prouvé sa capacité à exploiter les services d'une façon fiable et opérationnelle, contrastant avec les attentes de disponibilité d'AWS
