## Résumé de l'article

BBEdit est un éditeur de texte professionnel pour macOS. La version 16 introduit plus d'une centaine de nouvelles fonctionnalités, améliorations de performance et raffinements majeurs.

- Recherche de texte dans les images, y compris via grep multi-fichiers
- Sélection de schémas de couleur par projet et par Notebook, avec options personnalisables
- Nouveaux raccourcis et actions basées sur App Intents pour intégration à d'autres applications
- Temps de réponse réduits dans AI worksheet avec flux de résultats en direct
- Émulation de clavier vi pour la navigation et l'édition
- Mise à gratuite pour les clients BBEdit 15 (achetés après le 1er novembre 2025) ; mise à jour payante (29,99 $ ou 39,99 $ USD selon la version précédente)

## Discussion sur Hacker News (99 commentaires)

**Avis positifs** :
- Modèle commercial soutenable : prise de licence unique avec mises à jour optionnelles, sans abonnement, contrastant favorablement avec les pratiques de l'industrie moderne
- Outil polyvalent et puissant pour la manipulation de texte : possibilité d'étendre via scripts shell, Python, Rust ou autres outils, avec intégration fluide à l'interface
- Loyauté client exceptionnelle : logiciel natif macOS entretenu depuis 35 ans, mis à jour promptement à chaque évolution d'OS, avec politique généreuse de mise à jour pour les acheteurs récents
- Nouvelles fonctionnalités appréciées : émulation Vi/Vim, recherche de texte dans les images, feuilles de travail shell, support d'IA (Claude, OpenAI, Ollama), regroupement de documents sans titre
- Qualité et fiabilité éprouvées : feedback des utilisateurs de longue date attestant d'une excellence logicielle rare, sans compromis malgré la pression commerciale

**Avis négatifs** :
- Ensemble de features perçu comme incomplet : l'ajout de Vi/Vim est apprécié mais l'ensemble des mises à jour version 16 manque d'innovation majeure, ressentie comme 'plate' par certains utilisateurs
- Manque de certaines fonctionnalités modernes : absence de support multi-curseurs, interface jugée 'maladroite' et 'conventionnelle', pas d'intégration avec les APIs macOS récentes (widgets, Apple Intelligence, CloudKit avancé)
- Concurrence des outils plus modernes : les utilisateurs migrent vers Vim/Neovim, Emacs, VS Code ou des IDE JetBrains, avec BBEdit relégué à rôle de secours ou éditeur ponctuel
- Logiciel concurrent abandonné toujours promu : Yojimbo, bien que fonctionnel, n'a pas été mis à jour depuis 2020 et donne une impression d'abandon malgré les défenses d'usage comme logiciel 'fini'
- Pertinence questionnée pour les workflows modernes : cas d'usage moins clairs avec l'émergence des LLMs pour automatisation, alternatives spécialisées (Obsidian pour notes, VS Code pour développement) réduisant la niche de BBEdit
