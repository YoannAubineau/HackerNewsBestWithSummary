## Résumé de l'article

Le Libre Barcode Project est un ensemble de polices de caractères libres permettant de créer des codes-barres dans différents formats, avec ou sans texte sous le code. L'outil historique d'encodage Code 128 reste accessible à son URL originale pour préserver les liens existants.

- Les polices Libre Barcode permettent de générer des codes-barres scannables directement en utilisant des caractères
- Le projet supporte plusieurs formats de codes-barres via des polices dédiées
- Un encodeur Code 128 interactif est inclus et accessible pour prévisualiser les codes-barres générés
- Chaque format dispose d'une page dédiée avec des instructions d'utilisation détaillées

## Discussion sur Hacker News (36 commentaires)

**Avis positifs** :
- C'est un projet créatif et amusant qui pousse les limites du possible avec les polices de caractères (animations Bad Apple, codes-barres complexes)
- Le projet offre une solution pratique avec calcul automatique de sommes de contrôle pour certains formats comme EAN13, contrairement aux approches manuelles
- C'est une application originale et utile de technologies de polices existantes, avec des variantes de Code 128 bien documentées

**Avis négatifs** :
- Les polices de codes-barres ne sont pas la bonne approche : générer des images vectorielles ou bitmap avec la bonne résolution est plus fiable et tout aussi simple
- Les scanners éprouvent déjà des difficultés avec les codes-barres simples (problèmes de clavier émulé selon la langue), les polices amplifient ces problèmes
- Cette pratique, bien que courante industriellement, est une 'perversion' du design qui fonctionne mal au-delà de simples codes-barres (Code39 simple), devenant impraticale pour Code128 et les formats UPC/EAN
- La support Unicode est limité : les standards de codes-barres ne supportent généralement que l'ASCII, avec seulement quelques caractères ISO-8859-1 via des commutations complexes
