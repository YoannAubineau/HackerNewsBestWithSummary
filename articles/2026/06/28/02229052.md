## Résumé de l'article

Medium, plateforme de publication en ligne, a découvert un bug empêchant les utilisateurs polonais de taper la lettre Ś. Ce problème résultait de la collision de quatre circonstances historiques : l'utilisation en Pologne de la combinaison Alt+S pour les caractères accentués (héritage des claviers non-polonais importés pendant la Communisme), le mappage interne Windows de Alt+S vers Ctrl+Alt+S, et le code de Medium qui bloquait Ctrl+S pour empêcher le dialogue de sauvegarde du navigateur.

- Les claviers polonais des années 1980 utilisaient Alt comme modificateur pour accéder aux neuf diacritiques supplémentaires de la langue, faute de claviers localisés disponibles à l'époque du Communisme
- Windows mappait internement Alt+S (Alt droit) vers Ctrl+Alt+S pour compatibilité avec les anciens claviers single-Alt
- Le code de Medium bloquait Ctrl+S sans vérifier si Alt était pressée simultanément, interceptant ainsi Ctrl+Alt+S (la lettre Ś)
- La correction a consisté à modifier le filtre pour bloquer Ctrl+S seulement si Alt n'était pas pressée
- Ce bug illustre comment les conventions anglo-américaines en informatique continuent de pénaliser les utilisateurs de langues non-anglaises
